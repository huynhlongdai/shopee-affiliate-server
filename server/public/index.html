Cài đặt tài khoản

⚠️ Phải lấy cookie từ affiliate.shopee.vn (không phải shopee.vn thường):
  1. Mở affiliate.shopee.vn/tool/product-link và đăng nhập Affiliate.
  2. Nhấn F12 → chọn tab Console.
  3. Gõ lệnh sau rồi Enter, copy toàn bộ kết quả:
    document.cookie
  4. Dán vào ô Cookie bên trên rồi nhấn 💾 Lưu tài khoản.
Cách lấy đầy đủ hơn (DevTools → Application):
  1. Trên trang affiliate.shopee.vn, nhấn F12 → tab Application → Cookies → chọn https://affiliate.shopee.vn.
  2. Copy tất cả cookie (đặc biệt: SPC_EC, SPC_F, csrftoken, SPC_ST).

Hoặc: dùng chức năng Xuất JSON từ Chrome extension → Nhập JSON vào đây.
