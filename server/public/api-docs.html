📡 API Documentation

Shopee Affiliate Smart Link Server — dùng cho bot, n8n workflow, automation

🌐 Base URL:
🔑 Xác thực: Các endpoint có nhãn Auth yêu cầu header:
Authorization: Bearer <token>

Lấy token bằng cách gọi POST /api/auth/login trước.
🔐 Xác thực
POST /api/auth/login Đăng nhập, lấy JWT token
Request Body (JSON)
Tham sốLoạiBắt buộcMô tả
passwordstringRequiredMật khẩu đăng nhập
Ví dụ Request
curl -X POST '${BASE}/api/auth/login' \
  -H 'Content-Type: application/json' \
  -d '{"password":"your-password"}'
Response thành công
{
  "ok": true,
  "token": "eyJhbGci...",
  "mfaRequired": false
}
Response nếu 2FA bật
{
  "ok": true,
  "mfaRequired": true,
  "tempToken": "eyJhbGci..."
}
POST /api/auth/verify-mfa Xác thực mã 2FA, lấy JWT cuối
Request Body (JSON)
Tham sốLoạiBắt buộcMô tả
tempTokenstringRequiredToken tạm từ login
codestringRequiredMã 6 chữ số từ app 2FA
Ví dụ Request
curl -X POST '${BASE}/api/auth/verify-mfa' \
  -H 'Content-Type: application/json' \
  -d '{"tempToken":"eyJ...","code":"123456"}'
🍪 Đồng bộ Cookie
POST /api/sync-cookie Extension gửi cookie lên server
Request Body (JSON)
Tham sốLoạiBắt buộcMô tả
cookiestringRequiredChuỗi cookie Shopee Affiliate
accountNamestringOptionalTên tài khoản để nhận biết
Ví dụ
curl -X POST '${BASE}/api/sync-cookie' \
  -H 'Content-Type: application/json' \
  -d '{"cookie":"SPC_EC=...;SPC_F=...","accountName":"Tài khoản chính"}'
POST /api/sync-cookie-local Sync cookie nhanh từ local

Tương tự /api/sync-cookie — dùng cho đồng bộ thủ công từ máy local.

Request Body (JSON)
Tham sốLoạiBắt buộcMô tả
cookiestringRequiredChuỗi cookie
accountNamestringOptionalTên tài khoản
GET /api/sync-cookie Lấy cookie đang lưu trên server
Response
{
  "ok": true,
  "cookie": "SPC_EC=...;SPC_F=...",
  "accountName": "Tài khoản chính",
  "syncedAt": "2025-01-01T00:00:00.000Z"
}
⚡ Convert Link
POST /api/convert Auth  Convert Shopee links thành affiliate links
Request Body (JSON)
Tham sốLoạiBắt buộcMô tả
linksstring[]RequiredMảng URL Shopee cần convert
subIdsstring[]OptionalSub-ID để tracking (tối đa 5)
cookiestringOptionalCookie Shopee. Nếu bỏ trống, server tự dùng cookie đã sync từ extension
Ví dụ Request (n8n / bot)
curl -X POST '${BASE}/api/convert' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer eyJhbGci...' \
  -d '{
    "links": [
      "https://shopee.vn/product-name-i.123.456",
      "https://s.shopee.vn/abc123"
    ],
    "subIds": ["n8n", "bot"]
  }'
Response
{
  "ok": true,
  "mapping": {
    "https://shopee.vn/product-name-i.123.456": {
      "shortLink": "https://s.shopee.vn/xxxxx",
      "product": {
        "name": "Tên sản phẩm",
        "price": "299.000đ",
        "image": "https://...",
        "shopName": "Tên shop"
      }
    }
  }
}
Workflow n8n (HTTP Request node)
Method:  POST
URL:     ${BASE}/api/convert
Headers: Authorization = Bearer {{$node["Login"].json.token}}
         Content-Type  = application/json
Body:    {
           "links": {{ $json.shopeeLinks }},
           "subIds": ["n8n"]
         }

🤖 Hướng dẫn tích hợp n8n
Luồng workflow n8n đơn giản:
  1. HTTP Request — POST /api/auth/login → lưu token
  2. HTTP Request — POST /api/convert với header Authorization: Bearer {{token}}
  3. Set node — lấy mapping[link].shortLink từ response

Lưu ý: Cookie được tự động lấy từ extension đã sync — không cần truyền cookie trong request.
